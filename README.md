# Wild RSS

A podcast RSS feed validator.
Built as a side project to experiment with TanStack Start.

## Stack

- TanStack Start (React), Tailwind, React Aria
- Deployed on Cloudflare Workers

## Example

Podcasting 2.0 RSS feed: https://wildrss.com/?url=https%3A%2F%2Ffeeds.podcastindex.org%2Fpc20.xml
